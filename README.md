# Helwip — сервер двоетапної перевірки

Маленький сервер, який надсилає код підтвердження на email через Gmail
і перевіряє введений код. Потрібен, щоб код у сайті приходив реальним
листом, а не показувався прямо на екрані.

## Крок 1. Отримати App Password у Google

Звичайний пароль від Gmail тут не підійде — Google вимагає окремий
"пароль додатку" для програм, що відправляють листи через SMTP.

1. Увімкни двоетапну перевірку на самому Google-акаунті (якщо ще не
   увімкнена): https://myaccount.google.com/security
2. Перейди на https://myaccount.google.com/apppasswords
3. Створи новий пароль додатку (назви його, наприклад, "Helwip Server")
4. Google покаже 16-значний код виду `abcd efgh ijkl mnop` — скопіюй
   його, він знадобиться нижче. Це не той пароль, яким ти заходиш у
   пошту, тому його можна спокійно вставляти в налаштування сервера.

Зроби це з акаунту **shevchenko.myrosia@gmail.com** (або будь-якого з
двох — саме з нього листи й будуть надсилатись).

## Крок 2. Задеплоїти сервер на Render.com

1. Зареєструйся на https://render.com (можна через Google-акаунт)
2. Натисни **New +** → **Web Service**
3. Обери **Deploy an existing repository** — Render попросить
   підключити GitHub. Якщо репозиторію ще немає:
   - Створи новий репозиторій на https://github.com/new (наприклад
     `helwip-auth-server`)
   - Заливи туди файли з цієї папки (`server.js`, `package.json`)
   - Або, якщо не хочеш возитись з Git — напиши мені, і я підготую
     покрокову інструкцію із завантаженням прямо через веб-інтерфейс
     GitHub без командного рядка
4. У налаштуваннях сервісу на Render:
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Instance Type**: Free
5. У розділі **Environment** додай дві змінні:
   - `GMAIL_USER` = `shevchenko.myrosia@gmail.com`
   - `GMAIL_APP_PASSWORD` = той 16-значний код з Кроку 1
6. Натисни **Create Web Service** — Render збере і запустить сервер.
   За кілька хвилин отримаєш адресу виду
   `https://helwip-auth-server.onrender.com`

## Крок 3. Перевірити, що сервер працює

Відкрий у браузері `https://твій-сервер.onrender.com` — має з'явитись
напис "Helwip auth server is running."

## Крок 4. Підключити сервер до сайту

Дай мені адресу сервера, яку видав Render, і я онов`лю JS-код на
сайті helwip-store.html так, щоб він надсилав запити саме туди
замість локальної генерації коду.

## Важливо знати

- **Free-тариф Render "засинає"**, якщо сервером не користуються ~15
  хвилин — перший запит після паузи може виконуватись 20-30 секунд,
  поки сервер "прокидається". Для сімейного магазину це не критично,
  просто перше входження в систему після паузи буде повільнішим.
- Коди зберігаються **в пам'яті сервера**, тому якщо Render
  перезапустить сервіс (це буває на free-тарифі), усі незавершені
  запити коду обнуляться — просто треба буде запросити код ще раз.
- `GMAIL_APP_PASSWORD` — це не звичайний пароль від пошти, тому навіть
  якщо хтось його побачить, увійти в саму поштову скриньку через нього
  не можна, тільки надсилати листи від імені акаунту.
